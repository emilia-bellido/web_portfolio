# EBC Portfolio

A responsive, single-page personal portfolio website designed to showcase projects across web development, graphic design, video production, and content strategy. 

## 🚀 Features

* **Responsive Design:** Fully mobile-friendly layout built with Bootstrap 5.
* **Interactive Navigation:** Fixed top navbar with smooth scrolling to page sections.
* **Dynamic Portfolio Grid:** Utilizes Masonry layout for an optimized, cascading display of project cards.
* **Animated Skills Bars:** Visual representation of technical and digital skills using Bootstrap progress bars inside an accordion menu.
* **Multimedia Integration:** Embedded autoplaying video in the About section.
* **Downloadable Resume:** Direct link to download a PDF version of the resume.
* **Contact Form:** Clean, integrated contact layout for easy communication.

## 🛠️ Technologies Used

* **HTML5 & CSS3:** Semantic markup and custom styling.
* **Bootstrap 5.3.3:** CSS framework for the grid system, responsive behaviors, and UI components (accordions, cards, forms).
* **Masonry:** JavaScript grid layout library for the portfolio section.
* **Bootstrap Icons:** Scalable vector icons used throughout the site.
* **Google Fonts:** Typography styled with 'Inter' and 'Source Serif 4'.
